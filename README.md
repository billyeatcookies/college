## Instructions
1. install [obsidian](https://obsidian.md/)
2. get a git client:
   - pc, use [github desktop](https://desktop.github.com/)
   - android, use [mgit](https://f-droid.org/packages/com.manichord.mgit/)

clone this repository `https://github.com/billyeatcookies/college.git` and open the folder in obsidian.
